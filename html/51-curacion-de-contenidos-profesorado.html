5.1 Curación de contenidos para el profesorado

En este panel se recopilan recursos técnicos y pedagógicos útiles para preparar las sesiones y ampliar contenidos relacionados con el lenguaje procedimental en bases de datos.

Objetivo del panel

Este espacio de curación de contenidos permite al profesorado disponer de una colección organizada de recursos que facilitan la planificación y el desarrollo de la unidad didáctica.

  • Documentación oficial de MySQL
  • Tutoriales especializados
  • Recursos técnicos sobre Docker
  • Ejemplos de procedimientos almacenados
  • Materiales de apoyo para el aula

Los recursos han sido seleccionados para facilitar tanto la preparación de las sesiones como la resolución de posibles dudas técnicas durante el desarrollo del proyecto.
