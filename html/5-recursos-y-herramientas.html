5. Recursos y herramientas digitales

El desarrollo de esta unidad didáctica se apoya en diferentes herramientas digitales que permiten al alumnado trabajar con tecnologías reales del ámbito profesional del desarrollo de software.

Entorno técnico

  • Docker
  • MySQL
  • phpMyAdmin

Permiten crear un entorno real de trabajo con bases de datos.

Herramientas de desarrollo

  • VSCode / Antigravity
  • GitHub

Se utilizan para escribir, organizar y compartir el código SQL desarrollado.

Herramientas de aprendizaje

  • Vídeos educativos
  • Documentación oficial MySQL
  • Guías de práctica

Facilitan la comprensión de los conceptos y permiten ampliar conocimientos.

Curación de contenidos

  • Padlet
  • Symbaloo

Organizan recursos digitales para profesorado y alumnado.

Uso pedagógico de las herramientas

El uso de estas herramientas permite crear un ecosistema digital de aprendizaje donde el alumnado puede acceder a recursos, practicar con tecnologías reales y desarrollar su competencia digital dentro de un contexto profesional.

Además, la curación de contenidos facilita el acceso organizado a materiales de aprendizaje relevantes, tanto para el profesorado como para el alumnado.
